# GitHub Upload Anleitung - Quantum Avatar

## Schritt 1: Visual Studio Code öffnen
1. Öffne Visual Studio Code
2. Gehe zu File → Open Folder
3. Wähle `C:\QuantumAvatar\`

## Schritt 2: Git Repository initialisieren
```bash
git init
git add .
git commit -m "Initial Quantum Avatar System"
```

## Schritt 3: GitHub Repository verbinden
```bash
git remote add origin https://github.com/DEIN-USERNAME/quantum-avatar.git
git branch -M main
git push -u origin main
```

## Schritt 4: Oder über VS Code GUI
1. Ctrl+Shift+P → "Git: Initialize Repository"
2. Source Control Panel → "Publish to GitHub"
3. Wähle Repository Name: "quantum-avatar"
4. Alle Dateien committen und pushen
